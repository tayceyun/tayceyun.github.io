## 一、6124-5178 压力线分析

### 1.1 历史背景

6124-5178 压力线是 A 股市场最重要的长期技术压力线之一，连接了两个历史性高点：

| 高点 | 日期 | 点位 | 历史背景 |
|------|------|------|----------|
| 第一高点 | 2007年10月16日 | 6124.04 | 股权分置改革后的大牛市顶点 |
| 第二高点 | 2015年6月12日 | 5178.19 | 杠杆牛市顶点 |

这条压力线代表了 A 股市场近 18 年来的**长期下降趋势**，其突破具有重大技术意义。

### 1.2 数学推导

#### 线性回归计算

两点确定一条直线，采用线性方程：

```
y = kx + b

其中：
- y = 压力线点位
- x = 时间（以月为单位，或天数）
- k = 斜率
- b = 截距
```

#### 斜率计算

```
设：
- 点1 (x1, y1) = (2007-10-16, 6124.04)
- 点2 (x2, y2) = (2015-06-12, 5178.19)

时间跨度（月）：
x2 - x1 = (2015-6) - (2007-10) = 92 个月

斜率 k = (y2 - y1) / (x2 - x1)
       = (5178.19 - 6124.04) / 92
       = -945.85 / 92
       = -10.2809 点/月
```

#### 任意日期压力线位置

```python
def calculate_pressure_line(target_date):
    """
    计算指定日期的压力线位置
    
    公式：P = 6124.04 + (-10.2809) × 月数差
    """
    base_date = datetime(2007, 10, 16)
    months_diff = (target_date.year - base_date.year) * 12 + \
                  (target_date.month - base_date.month)
    
    pressure_position = 6124.04 + (-10.2809) * months_diff
    return pressure_position
```

#### 示例计算

| 日期 | 月数差 | 压力线位置 |
|------|--------|------------|
| 2024年1月 | 195 | 6124.04 - 10.28×195 = 4119.44 |
| 2025年1月 | 207 | 6124.04 - 10.28×207 = 3996.08 |
| 2025年12月 | 218 | 6124.04 - 10.28×218 = 3882.78 |

### 1.3 突破有效性判断规则

根据技术分析理论，趋势线突破需要满足以下条件才视为有效突破：

| 规则 | 条件 | 说明 |
|------|------|------|
| **3%法则** | 收盘价突破压力线幅度 ≥ 3% | 避免假突破 |
| **三日原则** | 连续3个交易日站稳突破位 | 确认突破持续性 |
| **成交量配合** | 突破时成交量放大 | 确认市场认可度 |
| **收盘价确认** | 以收盘价而非盘中高点计算 | 避免冲高回落的假信号 |

## 二、技术指标计算规范

### 2.1 MACD 指标

#### 计算公式

```
1. 快速 EMA（12日）：EMA12 = EMA(收盘价, 12)
2. 慢速 EMA（26日）：EMA26 = EMA(收盘价, 26)
3. DIF（差离值）：DIF = EMA12 - EMA26
4. DEA（讯号线）：DEA = EMA(DIF, 9)
5. MACD柱状图：MACD = (DIF - DEA) × 2
```

#### EMA 计算

```
EMA(今日) = α × 今日收盘价 + (1-α) × EMA(昨日)

其中：α = 2 / (周期 + 1)
- 12日EMA：α = 2/13 ≈ 0.1538
- 26日EMA：α = 2/27 ≈ 0.0741
- 9日EMA（用于DEA）：α = 2/10 = 0.2
```

#### 信号判断规则

| 信号 | 条件 | 含义 |
|------|------|------|
| **金叉** | DIF 上穿 DEA | 买入信号 |
| **死叉** | DIF 下穿 DEA | 卖出信号 |
| **零轴上金叉** | DIF > 0 且金叉 | 强势买入信号 |
| **零轴下死叉** | DIF < 0 且死叉 | 强势卖出信号 |
| **底背离** | 价格新低，MACD 不新低 | 反转买入信号 |
| **顶背离** | 价格新高，MACD 不新高 | 反转卖出信号 |

#### 多周期 MACD 共振

| 周期组合 | 信号强度 |
|----------|----------|
| 日线金叉 + 周线金叉 + 月线金叉 | 极强多头 |
| 日线金叉 + 周线金叉 | 强多头 |
| 仅日线金叉 | 中性偏多 |
| 日线死叉 + 周线死叉 | 强空头 |

### 2.2 RSI 指标

#### 计算公式

```
1. 计算价格变化：Δ = 今日收盘 - 昨日收盘
2. 分离涨跌：
   - 涨幅：Gain = max(Δ, 0)
   - 跌幅：Loss = max(-Δ, 0)
3. 计算平均涨跌幅（14日）：
   - 平均涨幅：AvgGain = SMA(Gain, 14)
   - 平均跌幅：AvgLoss = SMA(Loss, 14)
4. 相对强弱：RS = AvgGain / AvgLoss
5. RSI = 100 - (100 / (1 + RS))
```

#### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 周期 | 14日 | Welles Wilder 推荐值 |
| 超买阈值 | 70 | 高于此值视为超买 |
| 超卖阈值 | 30 | 低于此值视为超卖 |

#### 信号判断规则

| RSI 区间 | 状态 | 信号含义 |
|----------|------|----------|
| > 80 | 严重超买 | 强烈卖出信号 |
| 70-80 | 超买 | 警示卖出信号 |
| 50-70 | 偏强 | 多头区域 |
| 30-50 | 偏弱 | 空头区域 |
| 20-30 | 超卖 | 警示买入信号 |
| < 20 | 严重超卖 | 强烈买入信号 |

### 2.3 均线系统

#### 计算公式

```
MA(N) = (P1 + P2 + ... + PN) / N

其中：P = 收盘价，N = 周期
```

#### 常用均线周期

| 均线 | 周期 | 含义 |
|------|------|------|
| MA5 | 5日 | 短期趋势（周线） |
| MA10 | 10日 | 短期趋势（半月线） |
| MA20 | 20日 | 中短期趋势（月线） |
| MA60 | 60日 | 中期趋势（季线） |
| MA120 | 120日 | 中长期趋势（半年线） |
| MA250 | 250日 | 长期趋势（年线） |

#### 均线排列判断

| 排列类型 | 条件 | 市场含义 |
|----------|------|----------|
| **多头排列** | MA5 > MA10 > MA20 > MA60 | 强势上涨趋势 |
| **空头排列** | MA5 < MA10 < MA20 < MA60 | 强势下跌趋势 |
| **混乱排列** | 均线交织 | 震荡整理 |

#### 价格与均线关系

| 位置关系 | 条件 | 信号 |
|----------|------|------|
| 价格站上所有均线 | P > MA5 > MA10 > MA20 | 强多头 |
| 价格跌破所有均线 | P < MA5 < MA10 < MA20 | 强空头 |
| 价格位于均线之间 | 混合状态 | 观望/震荡 |

### 2.4 布林带

#### 计算公式

```
1. 中轨：MB = MA(收盘价, 20)
2. 标准差：σ = STD(收盘价, 20)
3. 上轨：UB = MB + 2σ
4. 下轨：LB = MB - 2σ
5. 带宽：BW = (UB - LB) / MB × 100%
```

#### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 周期 | 20日 | John Bollinger 推荐值 |
| 标准差倍数 | 2 | 覆盖约 95% 的价格波动 |

#### 信号判断规则

| 价格位置 | 条件 | 信号含义 |
|----------|------|----------|
| 触及上轨 | P ≥ UB | 短期超买，可能回调 |
| 中轨上方 | MB < P < UB | 多头区域 |
| 中轨附近 | P ≈ MB | 中性 |
| 中轨下方 | LB < P < MB | 空头区域 |
| 触及下轨 | P ≤ LB | 短期超卖，可能反弹 |

#### 带宽收窄/扩张

| 带宽变化 | 市场含义 |
|----------|----------|
| 带宽收窄 (< 10%) | 波动率下降，即将变盘 |
| 带宽扩张 (> 20%) | 波动率上升，趋势加速 |

## 三、多周期共振分析

### 3.1 理论基础

多周期共振分析基于**分形理论**和**道氏理论**：

1. **分形理论**：市场走势在不同时间周期呈现相似的结构特征
2. **道氏理论**：主要趋势、次要趋势和短期波动相互验证

当多个时间周期发出一致信号时，该信号的可靠性显著提升。

### 3.2 周期划分

| 周期 | K线周期 | 分析意义 |
|------|---------|----------|
| 短周期 | 日线 | 短期波动，入场时机 |
| 中周期 | 周线 | 中期趋势，持仓方向 |
| 长周期 | 月线 | 长期趋势，战略判断 |
| 超长周期 | 年线 | 历史趋势，牛熊判断 |

### 3.3 共振权重设计

| 周期 | 权重 | 理由 |
|------|------|------|
| 日线信号 | 1 | 短期波动，参考意义较低 |
| 周线信号 | 2 | 中期趋势，较为重要 |
| 月线信号 | 3 | 长期趋势，核心参考 |
| 年线信号 | 4 | 历史趋势，战略级别 |

### 3.4 共振评分规则

```python
# 共振评分计算
score = 0

# 技术面信号（权重 60%）
if daily_signal == "BULL": score += 1
if weekly_signal == "BULL": score += 2
if monthly_signal == "BULL": score += 3

# 压力线信号（权重 20%）
if deviation_pct > 3:  # 有效突破
    score += 3
elif deviation_pct > 0:  # 站上压力线
    score += 1
elif deviation_pct > -3:  # 接近压力线
    score += 0
else:  # 远离压力线
    score -= 1

# 成交量信号（权重 10%）
if volume_ratio > 1.5:  # 放量
    score += 1
elif volume_ratio < 0.7:  # 缩量
    score -= 1

# 资金面信号（权重 10%）
if north_money_inflow:  # 北向资金流入
    score += 1
if north_money_outflow:  # 北向资金流出
    score -= 1
```

### 3.5 信号强度判定

| 评分区间 | 信号强度 | 操作建议 |
|----------|----------|----------|
| ≥ 8 | 强多头 | 积极做多，仓位 70-80% |
| 5-7 | 中多头 | 适度做多，仓位 50-60% |
| 2-4 | 偏多 | 观望或轻仓，仓位 30-40% |
| -1 ~ 1 | 中性 | 观望，保持现有仓位 |
| -4 ~ -2 | 偏空 | 减仓观望，仓位 20-30% |
| ≤ -5 | 强空头 | 清仓或空仓 |

## 四、资金面分析

### 4.1 北向资金

#### 数据来源

通过港股通（沪港通、深港通）渠道的外资净买入/卖出数据。

#### 分析规则

| 指标 | 计算方法 | 信号判断 |
|------|----------|----------|
| 单日净买入 | 买入额 - 卖出额 | > 50亿为大幅流入 |
| 连续流入天数 | 统计连续净买入天数 | ≥ 3天视为持续流入 |
| 累计净买入（10日） | 近10日净买入之和 | > 200亿为积极信号 |

#### 信号权重

| 北向资金状态 | 权重 |
|--------------|------|
| 连续3日以上大幅流入 | +2 |
| 单日大幅流入（> 100亿） | +1 |
| 小幅波动 | 0 |
| 单日大幅流出（> 100亿） | -1 |
| 连续3日以上大幅流出 | -2 |

### 4.2 两融余额

#### 计算公式

```
融资余额变化 = 今日融资余额 - 昨日融资余额
两融余额 = 融资余额 + 融券余额
```

#### 分析规则

| 指标 | 信号判断 |
|------|----------|
| 融资余额持续增加 | 杠杆资金看多 |
| 融资余额持续减少 | 杠杆资金看空 |
| 融券余额增加 | 空头力量增强 |

## 五、K线形态识别

### 5.1 连阳/连阴

| 形态 | 判断条件 | 信号含义 |
|------|----------|----------|
| 连阳 ≥ 5天 | 连续5根及以上阳线 | 强势上涨 |
| 连阳 ≥ 7天 | 连续7根及以上阳线 | 极强势（可能短期超买） |
| 连阴 ≥ 5天 | 连续5根及以上阴线 | 弱势下跌 |
| 连阴 ≥ 7天 | 连续7根及以上阴线 | 极弱势（可能短期超卖） |

### 5.2 特殊形态

| 形态 | 判断条件 | 信号含义 |
|------|----------|----------|
| **阳包阴** | 阳线实体完全覆盖前一阴线 | 反转看涨 |
| **阴包阳** | 阴线实体完全覆盖前一阳线 | 反转看跌 |
| **锤头线** | 下影线长度 > 2倍实体 | 底部反转信号 |
| **倒锤头** | 上影线长度 > 2倍实体 | 顶部反转信号 |
| **十字星** | 实体占比 < 10% | 变盘信号 |

## 七、风险提示

1. **技术分析的局限性**：技术分析基于历史数据，无法预测突发事件（如政策变化、黑天鹅事件）
2. **指标滞后性**：所有技术指标都是基于历史数据计算，存在一定滞后
3. **假信号风险**：单一指标可能产生假信号，需多维度验证
4. **市场环境变化**：不同市场环境（趋势/震荡）适用不同的分析方法
5. **参数敏感性**：参数设置会影响信号生成，需根据实际情况调整
